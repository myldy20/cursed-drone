# Cursed Drone 0.12 — сквозной workflow

Cursed Drone — живой инструмент звуковых пространств: выбираем **место**, формируем четырёх **актёров**, пропускаем их через индивидуальные FX и завершаем пространство эффектами мастер-шины.

## Единый путь

1. **МЕСТО** — ландшафт, пять исполнительских макросов и уровни/mute четырёх актёров.
2. **АКТЁР** — источник, движок и характер выбранного актёра.
3. **FX** — четыре последовательных эффекта актёра; тип и параметры находятся в одном редакторе.
4. **МАСТЕР** — итоговый уровень, темп и четыре эффекта после сведения актёров.
5. **ПАМЯТЬ** — восемь пользовательских состояний, восстановление ландшафта, язык и fade.

## Управление

| Кнопка | Значение |
| --- | --- |
| D-pad | навигация и изменение значения; удержание ускоряет шаг |
| A | открыть, подтвердить, выполнить действие или заглушить актёра |
| B | назад/отмена; удержание — аварийный Kill |
| X | следующий раздел текущей страницы |
| Y | контекстная помощь |
| L / R | предыдущая / следующая страница |
| Select | fade итогового выхода |
| Start | быстрое меню |
| Start + Select | сохранить состояние и выйти |

## Место

X переключает **Ландшафт → Макросы → Актёры**. В разделе актёров Left/Right выбирает актёра, Up/Down меняет его уровень, A включает mute. Mute теперь имеет приоритет в аудиографе: рецепт ландшафта и старый хвост индивидуального FX не могут снова включить дорожку.

## Актёр

Переключатель актёров доступен прямо на странице. В основном режиме находятся активность, источник Ландшафт/Музыкальный, движок, частота и характер. В расширенном — 16 музыкальных моделей, MAIN/AUX/MIX/STEREO, встроенные и пользовательские Scala-строи, корневая нота, Euclidean-события и четыре строки модуляции с биполярной глубиной и ограниченной cross-modulation скорости.

## FX и мастер

У каждого актёра четыре последовательных FX. После сведения работают ещё четыре Master FX. Длинный Delay, Diffuser или Reverse Grains на мастере сохраняет атмосферный хвост при смене места.

## Память

Autosave продолжает последнее состояние. Восемь явных слотов хранятся как `memory-1.cdrone` … `memory-8.cdrone`. **Восстановить ландшафт** возвращает четырёх актёров к рецепту места, сохраняя Master FX.
