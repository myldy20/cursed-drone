# Установка Cursed Drone на NextUI

> Статус: **проверено на реальной TrimUI Brick с NextUI**. В ходе внешнего теста подтверждены запуск приложения, изображение, звук и управление.

Пакет для Knulli/PortMaster на NextUI устанавливать нельзя: NextUI использует собственный формат Tool Pak.

## Установка

1. Скачайте артефакт `cursed-drone-nextui-tg5040` из успешной сборки GitHub Actions.
2. GitHub может завернуть его во внешний ZIP. Внутри должен находиться архив `cursed-drone-nextui-tg5040-test.zip`.
3. Распакуйте внутренний архив **в корень SD-карты NextUI**.
4. Проверьте итоговый путь:

```text
Tools/tg5040/Cursed Drone.pak/launch.sh
```

5. Безопасно извлеките карту, вставьте её в консоль и откройте **Tools → Cursed Drone**.

Не кладите Pak в скрытую папку `.system`: NextUI заменяет её при обновлении.

## Данные и логи

Автосохранение:

```text
.userdata/tg5040/cursed-drone/autosave.cdrone
```

Лог запуска:

```text
.userdata/tg5040/logs/Cursed Drone.txt
```

Если приложение мгновенно возвращается в меню, пришлите этот лог.

## Проверенная конфигурация

- Устройство: TrimUI Brick
- Прошивка: NextUI
- Путь Pak: `Tools/tg5040/Cursed Drone.pak/`
- Результат: приложение запустилось и корректно работало

Другие устройства NextUI и другие platform-папки пока не проверены.

## Ограничения

Cursed Drone запускается как отдельное SDL-приложение. Стандартные функции NextUI — save states, auto-resume и единое внутриигровое меню — к нему не применяются.

Чтобы сохранить состояние и выйти из приложения, удерживайте **Start**, затем нажмите **Select**.
