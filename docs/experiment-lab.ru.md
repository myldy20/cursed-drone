# Cursed Drone Experimental 0.10 — Performance Lab

Эта сборка существует только в ветке `experiment/plaits-performance-lab`. Она не заменяет публичную v0.9 и не должна сливаться в `main`, пока не пройдёт проверку на TrimUI Brick.

## Упрощённое управление

| Кнопка | Обычная работа |
| --- | --- |
| D-pad | только перемещение по интерфейсу |
| L/R | уменьшить / увеличить выбранное значение |
| A | открыть, выполнить или подтвердить |
| B | назад или отмена; удерживать для аварийного Kill |
| X | следующий основной экран |
| Y | следующий актёр |
| Select | fade выхода |
| Start | открыть Performance Lab |
| Start + Select | сохранить и выйти |

На TrimUI Brick физические подписи A/B используются именно в этой логике, независимо от внутренней нумерации SDL.

## Performance Lab

Нажмите `Start`. D-pad влево/вправо меняет раздел, вверх/вниз — поле. L/R меняют значение. A выполняет действие. Y выбирает актёра. B закрывает Lab.

### ACTOR

- `ENGINE`: активирует оригинальный macro-oscillator actor;
- `MODEL`: выбирает один из 16 музыкальных движков;
- `OUTPUT`: `MAIN`, `AUX`, `MIX` или настоящий `STEREO`;
- `SCALA`: выбирает строй из встроенных или пользовательских `.scl`;
- `ROOT`: корневая MIDI-нота строя.

Пользовательские Scala-файлы кладутся в `curseddrone/conf/scales/`. При следующем запуске они появятся в списке.

### MOD

Четыре строки модуляции на актёра. Кнопка X выбирает строку.

- включение;
- форма: sine, triangle, sample-and-hold, random walk;
- назначение;
- скорость;
- биполярная глубина от −100% до +100%;
- предыдущий модулятор как источник изменения скорости;
- биполярная глубина cross-mod.

Для устойчивости модулятор может влиять только на скорость строки выше него. Так невозможно случайно собрать мгновенную цифровую петлю.

### EVENT

Euclidean-генератор событий:

- длина 1–32 шага;
- количество импульсов;
- поворот;
- вероятность;
- установка `Reverse Grains` в FX4.

Он запускает возбуждение музыкальных и физических движков, но не превращает основной экран в секвенсор.

### MORPH

- целевой ландшафт;
- плавный переход 0–100%;
- `COMMIT` применяет целевую сцену и обнуляет переход.

Параметры и уровни меняются плавно. Типы движков и цепочки эффектов переключаются около середины морфа, чтобы не удваивать DSP четырьмя дополнительными актёрами.

### STAGE

A открывает минимальный сценический экран:

- пять performance-макросов;
- morph;
- уровни и mute четырёх актёров;
- выходной fade.

B возвращает обычный интерфейс. Все подробные настройки остаются в Lab и не мешают живой работе.

## Reverse Grains

Два перекрывающихся обратных зерна читают общий кольцевой буфер с разными смещениями. Параметры:

- Mix;
- Grain — длительность и разброс;
- Feedback.

Эффект использует память конкретного FX-слота, а не создаёт дополнительный глобальный многомегабайтный буфер.

## Источник macro-oscillator

Ветка содержит зафиксированный upstream `pichenettes/eurorack` и использует оригинальный код Plaits и stmlib под лицензией MIT. Названия Cursed Drone нейтральны; проект не выдаёт производную работу за официальный продукт Mutable Instruments.
