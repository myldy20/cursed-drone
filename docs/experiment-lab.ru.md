# Cursed Drone Experimental 0.11 — Guided Workflow

Эта сборка существует только в ветке `experiment/v0.11-guided-workflow`. Она ставится рядом с публичной версией как **Cursed Drone Lab** и использует собственные сохранения.

## Единая логика

Путь пользователя теперь проходит через пять страниц:

1. **МЕСТО** — выбрать ландшафт, настроить его состояние и уровни четырёх актёров;
2. **АКТЁР** — настроить источник звука выбранного актёра;
3. **FX** — четыре последовательных эффекта актёра;
4. **МАСТЕР** — итоговая громкость, темп и четыре общих эффекта;
5. **ПАМЯТЬ** — восемь пользовательских состояний, восстановление ландшафта и системные настройки.

Термины используются последовательно: ландшафт — это место; в нём живут четыре актёра; каждый актёр идёт через свою цепочку FX; затем всё смешивается и проходит через Master FX.

## Управление

| Кнопка | Всегда означает |
| --- | --- |
| D-pad | выбрать поле или изменить его значение |
| A | открыть, применить или выполнить выбранное действие |
| B | назад или отмена; удерживать для аварийного Kill |
| X | следующий раздел текущей страницы |
| Y | контекстная помощь |
| L / R | предыдущая / следующая страница |
| Select | fade итогового выхода |
| Start | быстрое меню |
| Start + Select | сохранить последнее состояние и выйти |

Длительное удержание направления при изменении числового значения ускоряется в два, затем в пять раз. Это работает и для музыкального источника, событий и модуляции.

## Страница «Место»

Кнопка X переключает три раздела:

- **ландшафт** — A открывает список мест;
- **макросы** — вверх/вниз выбирают Материал, Активность, Напряжение, Дистанцию или Развитие; влево/вправо меняют значение;
- **актёры** — влево/вправо выбирают актёра, вверх/вниз меняют громкость, A включает или заглушает.

## Страница «Актёр»

X переключает **Основное** и **Расширенное**.

В основном разделе можно включить актёра, выбрать источник **Ландшафт / Музыкальный**, выбрать обычный движок и изменить характер звука. Повторный выбор `Ландшафт` возвращает только этого актёра к рецепту текущего места — менять весь ландшафт больше не нужно.

В расширенном разделе находятся:

- 16 моделей музыкального источника;
- основной, дополнительный, смешанный или стереовыход;
- стандартный и Scala-строй;
- корневая нота, показанная музыкальным именем, а не только MIDI-номером;
- Euclidean-события;
- четыре строки модуляции и безопасная cross-modulation.

## FX и Master FX

У каждого актёра остаются четыре эффекта. На странице Master добавлены ещё четыре общих эффекта после смешивания актёров. Длинный Delay или Diffuser на Master помогает сохранять хвост при смене ландшафта.

## Память и восстановление

Программа всегда продолжает последнее состояние через `autosave.cdrone`. Дополнительно есть восемь простых пользовательских слотов:

```text
conf/memory-1.cdrone ... conf/memory-8.cdrone
```

Действие **Восстановить ландшафт** возвращает четырёх актёров к исходному рецепту места, но сохраняет общую Master FX-цепочку.

## Что убрано из рабочего пути

Отдельные экраны Morph и Stage скрыты: они создавали больше терминов и режимов, чем пользы. Аудиодвижок и экспериментальные возможности сохранены, но доступны внутри понятного пути «Место → Актёр → FX → Мастер → Память».
