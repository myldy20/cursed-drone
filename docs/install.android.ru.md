# Установка Cursed Drone 0.13.0 на Android — preview

Android-сборка — сенсорная **ARM64 preview-версия** для Android 8.0 и новее. Звуковое ядро, места, эффекты и формат сессии совпадают с версиями для приставки и macOS, но интерфейс отдельный, полноэкранный и рассчитанный на горизонтальный экран телефона.

1. Скачай `cursed-drone-v0.13.0-android-arm64-preview.apk` из GitHub Release.
2. Разреши установку из браузера или файлового менеджера, которым открыт APK.
3. Установи и запусти приложение; при необходимости поверни телефон горизонтально.

Preview APK подписан отладочным ключом для ручной установки и пока не предназначен для Google Play. Autosave, восемь слотов и пользовательские Scala-файлы хранятся в приватных данных приложения. При удалении приложения они тоже удаляются, если Android не восстановит резервную копию.

Сенсорный workflow: прямые вкладки страниц, mute на карточках актёров, перетаскиваемые слайдеры и модальные сетки. На вкладке событий актёра доступны Trigger Now, Event Rate/FMAX, Euclidean и строй.
