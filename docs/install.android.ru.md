# Android-версия: alpha

Android-порт сохраняет существующий C++-аудиодвижок и SDL-интерфейс. Поверх них добавлены только полноэкранная Android-обвязка, нативная ARM64-сборка и адаптер сенсорного управления.

## Статус

Это **alpha-версия** для тестирования на реальных устройствах до публичного релиза.

- Android 8.0 и новее (`minSdk 26`)
- только ARM64 (`arm64-v8a`)
- ландшафтная ориентация
- наушники, USB-аудиоинтерфейсы и динамик телефона работают через SDL/Android Audio
- autosave и восемь слотов памяти хранятся в закрытой папке приложения

## Установка сборки из CI

1. Откройте запуск workflow `android` в GitHub Actions.
2. Скачайте артефакт `cursed-drone-android-debug`.
3. Распакуйте его и установите `app-debug.apk` на Android-устройство.
4. Android может попросить разрешить установку приложений из браузера или файлового менеджера, через который открыт APK.

Debug APK не предназначен для публикации в магазине, но подписан стандартным отладочным Android-ключом и устанавливается напрямую.

## Сенсорное управление

Исходный интерфейс инструмента 512×384 расположен слева. Справа находится отдельная сенсорная панель.

| Действие | Результат |
| --- | --- |
| Касание рабочей области | A / открыть / применить |
| Двойное касание рабочей области | Следующий раздел экрана |
| Горизонтальный или вертикальный свайп | D-pad: навигация и изменение значений |
| Удержание стрелки на панели | Повтор D-pad с исходным ускорением |
| PAGE− / PAGE+ | Предыдущая / следующая страница |
| BACK | Назад или отмена |
| SECTION | Следующий раздел текущей страницы |
| HELP | Контекстная помощь |
| MENU | Быстрое меню |
| FADE | Плавно открыть или закрыть выход |
| KILL | Немедленная остановка звука |

Внешние клавиатуры и геймпады продолжают работать через SDL.

## Локальная сборка

Понадобятся:

- Android Studio или Android command-line tools
- JDK 17
- Android SDK 35
- Android NDK `27.1.12297006`
- CMake 3.22.1
- Gradle 8.10.2 или совместимая версия

Клонируйте репозиторий вместе с сабмодулями и соберите APK:

```bash
git clone --recurse-submodules https://github.com/myldy20/cursed-drone.git
cd cursed-drone
gradle -p android :app:assembleDebug
```

Готовый APK появится здесь:

```text
android/app/build/outputs/apk/debug/app-debug.apk
```

Во время сборки Gradle загружает официальный Android Java-shim SDL2 и нативный исходный код SDL2 из зафиксированного тега `release-2.32.10`. Аудиодвижок Cursed Drone и сабмодуль Plaits компилируются непосредственно в APK.

## Известные ограничения alpha

- собирается только ARM64;
- пока нет Android App Bundle и настройки релизной подписи;
- сенсорная панель использует фиксированную ландшафтную компоновку и требует проверки на разных пропорциях экранов телефонов и планшетов;
- задержка звука и работа USB-аудио зависят от устройства и прошивки Android;
- обработку звонков, смены Bluetooth-устройств и горячего подключения аудиоинтерфейса нужно проверить на железе.
